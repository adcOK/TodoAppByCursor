# Cursorハンズオン用アプリ
## Todoアプリ（React + TypeScript）

### 概要
シンプルなTodoアプリを作成しながら、Cursorの主要機能を体験

### 対象
- エンジニア（初心者ではない）
- 所要時間：30-40分
- 環境準備：最小限


### 学べるCursor機能
- AIアシスタントとの対話によるコード生成
- コンポーネントの自動生成
- リファクタリング支援
- テストコードの自動生成
- エラーの自動修正

### 環境準備
```bash
# Node.jsがインストールされていること（v18以上推奨）
node --version
npm --version

# プロジェクト作成（5分）
npx create-react-app cursor-todo --template typescript
cd cursor-todo
npm start
```

### 実装内容（30分）
1. **基本Todo機能（10分）**
   - Todoアイテムの追加
   - Todoアイテムの削除
   - 完了状態の切り替え

2. **機能拡張（10分）**
   - フィルタリング（全て/未完了/完了）
   - ローカルストレージへの保存
   - アニメーション追加

3. **テスト追加（10分）**
   - ユニットテストの自動生成
   - コンポーネントテスト

### メリット
- ✅ 理解しやすい
- ✅ Cursorの機能を幅広く体験できる
- ✅ 完成形が見えやすい
- ✅ 実用的なスキルが身につく
