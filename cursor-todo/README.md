# Todoアプリ

React + TypeScriptで作成されたシンプルで機能的なTodoアプリケーションです。

## 機能

- ✅ Todoアイテムの追加
- ✅ Todoアイテムの削除
- ✅ 完了状態の切り替え
- ✅ フィルタリング（全て/未完了/完了）
- ✅ ローカルストレージへの自動保存
- ✅ モダンなUIデザインとアニメーション

## 技術スタック

- React 19.2.0
- TypeScript 4.9.5
- CSS3（アニメーション含む）
- React Testing Library（テスト用）

## セットアップ

### 必要な環境

- Node.js v18以上
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install
```

## 実行方法

### 開発サーバーの起動

```bash
npm start
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてアプリを確認できます。

### ビルド

```bash
npm run build
```

本番環境用の最適化されたビルドを作成します。

### テストの実行

```bash
npm test
```

テストスイートを実行します。

## プロジェクト構造

```
src/
├── components/          # Reactコンポーネント
│   ├── TodoInput.tsx   # Todo入力コンポーネント
│   ├── TodoItem.tsx    # Todoアイテムコンポーネント
│   ├── TodoList.tsx    # Todoリストコンポーネント
│   └── __tests__/      # コンポーネントテスト
├── hooks/              # カスタムフック
│   └── useLocalStorage.ts  # ローカルストレージフック
├── types.ts            # TypeScript型定義
├── App.tsx             # メインアプリケーションコンポーネント
└── App.test.tsx        # アプリケーションテスト
```

## 使い方

1. **Todoの追加**: 入力フィールドにTodoの内容を入力し、「追加」ボタンをクリック
2. **完了状態の切り替え**: Todoアイテムのチェックボックスをクリック
3. **Todoの削除**: Todoアイテムの「×」ボタンをクリック
4. **フィルタリング**: 「全て」「未完了」「完了」ボタンで表示を切り替え

## 特徴

- **ローカルストレージ**: ブラウザを閉じてもTodoが保存されます
- **レスポンシブデザイン**: モバイルデバイスでも快適に使用できます
- **アニメーション**: スムーズなフェードイン/フェードアウトアニメーション
- **型安全**: TypeScriptによる完全な型安全性
