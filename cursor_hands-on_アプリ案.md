# Cursorハンズオン用アプリ案

## 対象
- エンジニア（初心者ではない）
- 所要時間：30-40分
- 環境準備：最小限

---

## 案1: Todoアプリ（React + TypeScript）⭐ 推奨

### 概要
シンプルなTodoアプリを作成しながら、Cursorの主要機能を体験

### 学べるCursor機能
- AIアシスタントとの対話によるコード生成
- コンポーネントの自動生成
- リファクタリング支援
- テストコードの自動生成
- エラーの自動修正

### 環境準備
```bash
# Node.jsがインストールされていること（v18以上推奨）
node --version
npm --version

# プロジェクト作成（5分）
npx create-react-app cursor-todo --template typescript
cd cursor-todo
npm start
```

### 実装内容（30分）
1. **基本Todo機能（10分）**
   - Todoアイテムの追加
   - Todoアイテムの削除
   - 完了状態の切り替え

2. **機能拡張（10分）**
   - フィルタリング（全て/未完了/完了）
   - ローカルストレージへの保存
   - アニメーション追加

3. **テスト追加（10分）**
   - ユニットテストの自動生成
   - コンポーネントテスト

### メリット
- ✅ 理解しやすい
- ✅ Cursorの機能を幅広く体験できる
- ✅ 完成形が見えやすい
- ✅ 実用的なスキルが身につく

## 案2: REST API（Node.js + Express）

### 概要
シンプルなREST APIを作成し、バックエンド開発でのCursor活用を体験

### 学べるCursor機能
- APIエンドポイントの自動生成
- エラーハンドリングの追加
- バリデーションロジックの生成
- ドキュメント生成

### 環境準備
```bash
# Node.jsがインストールされていること
node --version
npm --version

# プロジェクト作成（2分）
mkdir cursor-api && cd cursor-api
npm init -y
npm install express
npm install -D @types/express @types/node typescript ts-node nodemon
```

### 実装内容（35分）
1. **基本API（15分）**
   - GET /api/users - ユーザー一覧取得
   - POST /api/users - ユーザー作成
   - GET /api/users/:id - ユーザー取得
   - DELETE /api/users/:id - ユーザー削除

2. **機能拡張（10分）**
   - バリデーション追加
   - エラーハンドリング
   - ロギング機能

3. **テスト追加（10分）**
   - APIテストの自動生成
   - Postmanコレクション生成

### メリット
- ✅ バックエンド開発での活用方法が学べる
- ✅ 実務でよく使うパターンを体験
- ✅ 環境準備が非常に簡単

## 補足：Cursorの主要機能紹介ポイント

どの案でも以下の機能を紹介すると良いでしょう：

1. **Cmd/Ctrl + K**: インライン編集
2. **Cmd/Ctrl + L**: チャットでの対話
3. **コード生成**: 自然言語からコード生成
4. **リファクタリング**: コード改善の提案
5. **エラー修正**: 自動的なエラー修正
6. **テスト生成**: テストコードの自動生成
7. **ドキュメント生成**: コメントやREADMEの生成
